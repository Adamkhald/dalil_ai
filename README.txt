========================================================================
                       DALIL AI - USER GUIDE
========================================================================

Thank you for using Dalil AI!

This folder contains the full source code of the Dalil AI research platform.

HOW TO RUN
------------------------------------------------------------------------
Method 1: The Launcher (Recommended)
   Simply double-click the file named:
   [ run_windows.bat ]

   This will automatically find Python on your computer and launch the app.

Method 2: Command Line
   Open a terminal/command prompt in this folder and type:
   python main.py

REQUIREMENTS
------------------------------------------------------------------------
To run this application, you need:
1. Python 3.8 or higher installed.
2. The necessary libraries installed.

If the app closes immediately or doesn't start, run this command to 
install the dependencies:

   pip install -r requirements.txt

INCLUDED MODULES
------------------------------------------------------------------------
* Scikit-Learn (Machine Learning)
* PyTorch (Deep Learning / Research)
* TensorFlow (Deep Learning / Production)
* MediaPipe (Computer Vision)
* RL Studio (Reinforcement Learning)

If you don't use a specific module (e.g. TensorFlow), you can uninstall it
to save space. The app will automatically detect it is missing and just
disable that tab.
